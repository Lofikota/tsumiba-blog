FXを始めようとして最初の壁になるのがこれだ。数字が動き続ける画面を見ても、どこを見ればいいか分からない。

## FXチャートで最初に覚えること：3つだけ

チャートの読み方には無数の手法がある。だが初心者が最初に覚えるべきは3つだけだ。

1. **ローソク足** — 価格の動きを視覚化したもの
2. **トレンド** — 相場の大きな方向性
3. **サポート・レジスタンスライン** — 価格が反転しやすい水準

この3つを理解してからインジケーター（[MACD](/blog/fx-technical-indicator/)や[RSI](/blog/fx-technical-indicator/)）に進む。順番を間違えると、何を見ているか分からないまま取引してしまう。

## ローソク足の読み方

### ローソク足とは何か

ローソク足は、一定期間の価格変動を1本の「棒」で表したものだ。

```
         ┃  ← 上ヒゲ（高値）
┌────────┐
│        │  ← 実体（始値〜終値）
│  陽線  │
│（緑/白）│
└────────┘
         ┃  ← 下ヒゲ（安値）
```

1本のローソク足に含まれる情報は4つ：

| 要素 | 意味 |
|------|------|
| **始値（Open）** | その時間帯の最初の価格 |
| **終値（Close）** | その時間帯の最後の価格 |
| **高値（High）** | その時間帯の最高値 |
| **安値（Low）** | その時間帯の最安値 |

### 陽線と陰線

- **陽線**（緑・白）：終値 ＞ 始値。その時間帯に価格が上昇した
- **陰線**（赤・黒）：終値 ＜ 始値。その時間帯に価格が下落した

FXチャートでよく使う時間足は1分・5分・15分・1時間・4時間・日足など。同じ[ドル円](/blog/fx-best-time/)でも「1時間足で見ると上昇トレンド、15分足で見ると下落中」という状況はよくある。まず**日足（1日=1本）から全体像を把握する**のが基本だ。

### よく出るローソク足パターン

**長い上ヒゲ**：価格が上がったが押し戻された → 上値の重さのサイン
**長い下ヒゲ**：価格が下がったが戻された → 下値の固さのサイン
**コマ足（実体が小さい）**：買い・売りが拮抗している → 方向感のなさのサイン

ただし1本のローソク足で判断するのは危険だ。必ず複数本の流れで判断する。

## トレンドの判断方法

### 3種類のトレンド

相場には必ず方向性がある。

1. **上昇トレンド**：高値と安値が切り上がっている（高値↗ 安値↗）
2. **下降トレンド**：高値と安値が切り下がっている（高値↘ 安値↘）
3. **レンジ（横横）**：高値と安値がほぼ一定の範囲内で推移している

FXで最もやりやすいのは**トレンドに乗る取引**だ。「上昇トレンド中に押し目（一時的な下落）で買う」「下降トレンド中に戻り（一時的な上昇）で売る」が基本的な考え方になる。

### トレンドラインの引き方

上昇トレンドなら**安値を結んだ線**（サポートライン）を引く。下降トレンドなら**高値を結んだ線**（レジスタンスライン）を引く。

```
        /高値3
      /
   /高値2
 /
高値1

安値1
  \
   \安値2
     \
      \安値3
```

このトレンドラインを価格が上から下に割った（ブレイク）とき、トレンド転換のサインになることが多い。

## サポートライン・レジスタンスラインとは

### 価格が「反転しやすい水準」がある

相場には「この価格になると買いが入りやすい（下がりにくい）」「この価格になると売りが増える（上がりにくい）」という水準がある。

- **サポートライン（支持線）**：価格が下がってくると反転して上昇しやすい水準
- **レジスタンスライン（抵抗線）**：価格が上がってくると反転して下落しやすい水準

過去に何度も価格が反転した水準は、多くのトレーダーが意識しているため、また同じ水準で反応が出やすい。

### なぜサポート・レジスタンスが機能するのか

FX市場には世界中のトレーダーが参加している。「[ドル円](/blog/fx-best-time/)が150円の節目」「過去の高値・安値ライン」など、誰もが意識しやすい価格帯では自然と売買が集中する。この集合的な意識がサポート・レジスタンスを機能させる。

### ブレイクアウト

サポートラインを価格が下に抜けたとき（サポートブレイク）は「それ以上下落するサイン」として見る。逆にレジスタンスを上に抜けたとき（レジスタンスブレイク）は「それ以上上昇するサイン」だ。

ただしブレイクアウトは「だまし」（一時的に抜けてすぐ戻る）も多い。1本のローソク足が抜けただけでなく、複数本にわたってブレイクが続いているか確認する習慣が大切だ。

## 初心者がよく使うインジケーター3選

ローソク足とトレンドが分かったら、補助的にインジケーターを使う。ただし最初は多く使いすぎないことが重要だ。

### 1. 移動平均線（MA）

一定期間の価格の平均を線で表したもの。

- **25日移動平均線**：約1ヶ月の平均価格
- **75日移動平均線**：約3ヶ月の平均価格

価格が移動平均線の上にあれば上昇トレンド、下にあれば下降トレンドの目安になる。2本の移動平均線が交差する「ゴールデンクロス（買いサイン）」「デッドクロス（売りサイン）」も基本的な判断軸だ。

### 2. RSI（相対力指数）

相場の「買われすぎ・売られすぎ」を0〜100の数値で示す。

- **70以上**：買われすぎ（反落の可能性）
- **30以下**：売られすぎ（反騰の可能性）

[RSI](/blog/fx-technical-indicator/)だけで取引判断するのは危険だが、「強いトレンドの中でRSIが70を超えた」というような複合的な判断材料として使う。

### 3. ボリンジャーバンド

移動平均線を中心に、価格の変動幅を±2σの帯で表したもの。

- 価格がバンドの上限（+2σ）を超えた：買われすぎの可能性
- 価格がバンドの下限（-2σ）を超えた：売られすぎの可能性
- バンドが狭くなってきた（スクイーズ）：大きな動きの前兆の可能性

インジケーターはあくまで補助ツールだ。ローソク足とトレンドを基本に、インジケーターで確認するという順番で使う。

## チャートを見るときの実践的な手順

初心者が取引前にチャートを確認する手順をまとめる。

**Step 1：日足で大きなトレンドを確認する**

まず日足チャートを開いて、過去3ヶ月〜6ヶ月の動きを見る。上昇・下降・レンジのどれかを確認する。

**Step 2：4時間足で中期的なトレンドを確認する**

日足のトレンドが確認できたら、4時間足を開いてより細かい動きを確認する。日足と4時間足が同じ方向を向いているときは比較的トレンドが強い。

**Step 3：1時間足でエントリーポイントを探す**

実際にどこで買う・売るかは1時間足で判断する。サポートライン近くまで価格が来たらエントリーを検討する、というイメージだ。

**Step 4：[損切り](/blog/fx-shoshinsha-guide/)ラインを決めてから注文する**

どこで[損切り](/blog/fx-shoshinsha-guide/)するかを決めてからエントリーする。「[損切り](/blog/fx-shoshinsha-guide/)ラインを決めずに入ったら損が膨らんで身動きが取れない」という状況は初心者に多い。損切りを設定することが[リスク管理](/blog/fx-dollar-yen-ugoki/)の基本だ（[FXロスカットの仕組みとリスク管理](/blog/fx-leverage-risk-guide/)も参照）。

## よくある質問

**Q. チャートを読めるようになるまでどのくらいかかりますか？**

A. ローソク足とトレンドの基本は1週間で覚えられます。ただし「読める」と「稼げる」は別です。[デモ口座](/blog/fx-demo-koza-osusume/)で最低1〜3ヶ月練習してから本番取引に移ることをすすめます。

**Q. どの時間足を使えばいいですか？**

A. 初心者は日足→4時間足→1時間足の順に確認する「マルチタイムフレーム分析」から始めるのがおすすめです。1分足・5分足だけを見るのは情報が多すぎて判断しにくいです。

**Q. テクニカル分析とファンダメンタルズ分析はどちらが重要ですか？**

A. 短期取引（デイトレード・[スキャルピング](/blog/fx-small-start-guide/)）はテクニカル中心、中長期取引はファンダメンタルズも重要になります。初心者はまずテクニカルの基礎を固めてから、[経済指標](/blog/fx-dollar-yen-ugoki/)の影響を学ぶ順番がおすすめです。

**Q. インジケーターはいくつ使えばいいですか？**

A. 最初は1〜2個に絞ることをすすめます。多く使うほど「どれに従えばいいか」が分からなくなります。移動平均線だけ、またはRSIだけで取引判断する期間を作るのが上達の近道です。

## まとめ

FXチャートの基本は3つだ。

1. **ローソク足**：始値・終値・高値・安値で価格変動を把握する
2. **トレンド**：高値・安値の切り上げ・切り下げで方向性を判断する
3. **サポート・レジスタンス**：過去に反転した水準を意識する
