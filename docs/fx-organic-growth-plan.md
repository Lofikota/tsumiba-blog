# FX SEO / X オーガニック運用計画

最終更新: 2026-05-10

## 方針

FX は Google Search Ads の主戦場にしない。既存ブログ資産では、広告審査リスクを取りに行くより、SEO と X の自然流入で育てる方が筋がよい。

```text
X投稿
→ FX比較・レビュー記事
→ LINEの interest_fx タグ
→ 案件別クリック後フォロー
→ ASP成果確認
```

## 理由

- 既存記事に `fx-kouza-hikaku` / `dmm-fx-review` / `jfx-review` / `fxtf-review` / `matsui-fx-review` / `fx-shoshinsha-guide` があり、内部リンククラスターを作りやすい。
- FX は高単価だが、広告では複雑な投機的金融商品ポリシーに触れやすい。
- X では体験談、失敗談、少額開始、リスク理解の文脈で自然に接点を作れる。
- LINE 側には `interest_fx` と案件別 click follow-up があり、読者の迷い潰しまで接続できる。

## SEOの狙い

優先する検索意図:

| 優先 | 意図 | 例 | 役割 |
|---:|---|---|---|
| 1 | 初心者の不安解消 | FX 初心者 やめとけ / FX 少額 始め方 | LINE登録・比較記事への入口 |
| 2 | 会社員の実務 | FX 確定申告 やり方 会社員 / FX 副業 バレる | 税金・副業記事との回遊 |
| 3 | 口座比較 | FX口座 初心者 おすすめ / DMM FX 評判 | 案件クリック |
| 4 | 取引前の安全確認 | FX レバレッジ 何倍 初心者 / ロスカット 仕組み | 信頼形成 |

## 記事キュー

`data/keyword-queue.json` では、FXオーガニック用の `pending` を優先して処理する。

優先順:

1. `fx-kakuteishinkoku-guide`
2. `fx-yametoke-reason`
3. `fx-small-start-guide`
4. `fx-leverage-risk-guide`
5. `fx-company-barenai`

各記事は、公開前に次を必ず入れる。

- 冒頭のPR表記
- 元本割れ、レバレッジ、ロスカット、預け入れ資金を上回る損失可能性の説明
- 公式情報への参照
- 「必ず儲かる」「誰でも稼げる」「安全に増える」系の禁止表現を避ける
- 末尾で `fx-kouza-hikaku` と LINE の FX導線へ自然に接続する

## X運用

FX投稿は売り込みではなく、不安解消と経験談から入る。

週次配分:

| 曜日 | 投稿テーマ | 主導線 |
|---|---|---|
| 月 | FX初心者の誤解・やめとけ論 | `fx-shoshinsha-guide` |
| 水 | 田中蓮の少額体験・失敗談 | `fx-kouza-hikaku` |
| 金 | 口座比較・条件整理 | `dmm-fx-review` / `jfx-review` / `fxtf-review` |
| 土 | 税金・会社員の注意点 | `fx-kakuteishinkoku-guide` |

投稿の原則:

- 「稼げる」より「何で失敗しやすいか」を先に出す。
- CTAは毎回入れず、3〜4本に1本だけブログへ送る。
- ハッシュタグは多くても2つ。例: `#FX初心者` `#お金の勉強`
- キャンペーンや条件は公開前に公式・ASP側で確認する。

## 計測

見る順番:

1. GSC: FX記事の表示回数、4〜20位クエリ、CTR
2. X: プロフィールクリック、URLクリック、保存/リポスト
3. ブログ: FX記事間の回遊、CTAクリック
4. LINE: `interest_fx` 付与数、FXリンククリック
5. ASP: 案件別クリックと成果

週次SEO改善では、FX記事の4〜20位クエリを優先してタイトル、冒頭、H2、内部リンクを直す。

## やらないこと

- FX Search Ads の新規出稿
- FX広告文の作成
- 断定的な収益訴求
- 公式確認なしのキャンペーン訴求
- 秘密情報、ASP管理画面情報、広告アカウント情報のMarkdown保存
